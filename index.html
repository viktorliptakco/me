Okej, trochu to rozvediem...

Programovanie študujem ako samouk. K takémuto rozhodnutiu ma viedla potreba kariérneho nakopnutia. Chcel som sa začať venovať niečomu čo ma baví, čomu budem rozumieť a čo má skutočnú perspektívu. IT zameranie je pre mňa vyhovujúce vo všetkých bodoch. Začiatky, samozrejme, boli a stále sú náročné.

Svoj prvý kurz som si zapísal v októbri 2022 na kurzovom portále Skilmea. Niekoľko som ich úspešne absolvoval a na ďalších pracujem. Okrem toho sa učím z najrôznejších zdrojov, napríklad videotutoriály na YT, rôzne blogy, ale aj pomocou AI - ChatGPT. Popri tvorení webu som začal objavovať možnosti grafických editorov a tiež kúzelných slov "how to...", ktoré sa dajú využiť prakticky pri každom probléme, nielen programovacom.

Kurzy, ktoré som absolvoval:

Webrebel 1: HTML, CSS & JavaScript

Rozsiahly kurz o základoch HTML5, CSS3, JavaScripte, JQuery, responzívnom dizajne a daľších témach.

Základy programovania a OOP

Princípy programovania vysvetlené v jazyku PHP. Objektovo orientované programovanie sprostredkované ľudskou rečou.

CSS preprocesory: Sass

CSS preprocesory zefektívňujú prácu s CSS3. Kódu je menej, je prehľadnejší a píše sa rýchlejšie.

Git a GitHub Základy

Git si pamätá históriu vývoja projektu, robí zálohy, ku ktorým dá kedykoľvek vrátiť a dovoľuje viacerým naraz pracovať s tými istými súbormi.

Rob hry v JavaScripte

Tvorba hry v čistom JavaScripte pomocou ktorej sú vysvetlené premenné, funkcie, cykly, podmienky a najzákladnejšie princípy game developmentu.

Kurzy, na ktorých makám:

Webrebel 2: PHP a MySQL

Programovanie dynamických webových stránok v programovacom jazyku PHP, ktoré komunikujú s databázou MySQL.

JavaScript a ES6

Kurz popisuje moderný JavaScript. Približuje používane JS v prehliadači.

Vue.js + single page aplikácie

Je reaktívny framework, ktorý pomáha programovať svižné UI. Zo statického HTML kódu spraví živú šablónu.

Úvod do UX/UI: Dizajn pre obrazovky

V kompaktnej forme teoretické základy UI, UX a interakčného dizajnu. Všetko od pixelov, cez wireframy až po prácu s pozornosťou užívateľa.

SQL databázy: MySQL a SQLite

Základy komunikácie s databázou. Databáza drží údaje, dovolí pridávať nové, upravovať a mazať existujúce a vyberať špecifické údaje na základe podmienok.

Game Development v Unity

Komplexná 3D hra od návrhu až po jej zostrojenie. Zahřňa minikurz programovacieho jazyka C# a vývoj hry v hernom engine Unity.
